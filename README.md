# Chat with bots

```
This is a chat with several bots that randomly generate messages. Built on technology websocket.
```

### Fisrt setup project
```
yarn install
```

### Second run project
```
yarn serve
```

### Thirt start server
```
node server
```

### Open localhost: http://127.0.0.1:8080/

### Run your unit tests
```
yarn test:unit
```
